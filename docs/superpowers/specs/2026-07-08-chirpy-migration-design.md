# Chirpy Theme Migration Design

**Date:** 2026-07-08
**Goal:** Migrate dchavezf.github.io from Jekyll minima to Chirpy theme, gaining sidebar navigation, dark/light mode, built-in search, TOC, categories/tags, and a modern professional layout while preserving all existing content and brand identity.

## Approach

Use the official `chirpy-starter` template as the base. Migrate all existing content into Chirpy's structure (`_tabs/`, `_posts/`, collections). Override Chirpy's SCSS with the site's existing design tokens (teal #0f9b8e, Inter/JetBrains Mono, custom radii and shadows). Deploy via GitHub Actions to GitHub Pages.

## Deployment

### GitHub Actions
The chirpy-starter includes `.github/workflows/pages-deploy.yml`:
1. Builds with `bundle exec jekyll build`
2. Deploys to GitHub Pages on push to `main`

### GitHub Pages Settings
- Settings → Pages → Source: **GitHub Actions** (not "Deploy from branch")
- Workflow handles everything automatically

### Files Removed
- `_includes/head.html` — Chirpy handles head, Mermaid, SEO natively
- `assets/css/style.scss` — replaced by `jekyll-theme-chirpy.scss`

### Files Preserved
- `voice_and_tone.md`, `AGENTS.md`, `CLAUDE.md` (updated for Chirpy references)
- `projects/*.md` (adapted with Chirpy frontmatter)
- `docs/adr/*.md` (preserved as-is)
- `_posts/*.md` (adapted with categories/tags)

## Features Gained

| Feature | Before (minima) | After (Chirpy) |
|---------|-----------------|----------------|
| Navigation | Header links | Sidebar tabs + search |
| Color mode | Light only | Dark/Light with toggle |
| Search | None | Built-in full-text search |
| TOC | None | Auto-generated per post |
| Categories | None | Hierarchical, auto-generated pages |
| Tags | None | Trending tags, auto-generated pages |
| Mermaid | Custom `_includes/head.html` | Native support |
| PWA | None | Built-in installable app |
| SEO | jekyll-seo-tag plugin | jekyll-seo-tag + enhanced meta |
| Pagination | None | Built-in (10 per page) |
